# Riptide Shield Fix

Download on [Modrinth](https://modrinth.com/project/pu0BaLmL)

Fixes equipped shields covering up two thirds of your screen whenever you use a riptide trident
([MC-127970](https://bugs.mojang.com/browse/MC/issues/MC-127970)).

This works for all items, not just shields, but shields are the most common thing to keep in your offhand,
and are the most egregious example of the bug.

This mod is also built with compatibility in mind, and should work with modded trident variants.
